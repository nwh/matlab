# Matlab Tools

This directory contains tools to automatically download and configure external
Matlab tools.

## Included tools

* CVX: <http://cvxr.com/>
* JSONLab: <http://iso2mesh.sourceforge.net/cgi-bin/index.cgi?jsonlab>

## Usage

From command line:

```
$ make
```
